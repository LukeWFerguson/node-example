# node-example

## Running Locally

1) Make sure you have [Node](https://nodejs.org/en/) installed.
2) Run `npm install`.
3) Run `node server.js`.
4) Visit http://localhost:8080/ to view the application.

> Reference: https://nodejs.org/en/docs/guides/getting-started-guide/
